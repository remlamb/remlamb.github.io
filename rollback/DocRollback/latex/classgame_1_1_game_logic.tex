\doxysubsection{Detailed Description}
Represents the game logic of the application, managing physics simulation, player interaction, and input handling. 

The \doxylink{classgame_1_1_game_logic}{Game\+Logic} class initializes and updates the game world, manages player input, handles collision detection, and controls game state transitions. It also creates platforms and ropes within the game world and updates their associated colliders.

Member Variables\+:
\begin{DoxyItemize}
\item world\+\_\+\+: The physics world for simulating game physics.
\item rollback\+\_\+manager\+: Manages game state rollback for network synchronization.
\item network\+\_\+logic\+: Manages networking logic and communication.
\item player\+\_\+manager\+: Manages player entities and their interactions with the game world.
\item inputs\+: Stores the current frame inputs for all players.
\item last\+\_\+inputs\+: Stores the inputs from the previous frames for rollback purposes.
\item network\+\_\+events\+: Queue to store incoming network events.
\item colliders\+\_\+\+: Vector of collider structures representing physics objects and their colliders.
\item client\+\_\+player\+\_\+nbr\+: ID of the local client player.
\item current\+\_\+game\+\_\+state\+: Enum representing the current state of the game.
\end{DoxyItemize}

Member Functions\+:
\begin{DoxyItemize}
\item Rollback\+: Rolls back the game state to match the provided \doxylink{classgame_1_1_game_logic}{Game\+Logic} instance.
\item Set\+Player\+Input\+: Sets the input for a specific player.
\item Compute\+Checksum\+: Computes a checksum for the current game state.
\item Register\+Network\+Logic\+: Registers the \doxylink{class_network_logic}{Network\+Logic} instance for networking purposes.
\item Send\+Frame\+Confirmation\+Event\+: Sends frame confirmation events to confirm synchronized frames.
\item On\+Frame\+Confirmation\+Received\+: Handles frame confirmation events received from the network.
\item Init\+: Initializes the game environment, including creating platforms and ropes.
\item On\+Input\+Received\+: Handles input events received from the network.
\item Update\+: Updates the game logic, processes inputs, and advances the game state.
\item De\+Init\+: Deinitializes the game environment.
\item Manage\+Input\+: Manages player inputs and sends input events to the network.
\item Update\+Gameplay\+: Updates player actions and game physics.
\item Create\+Platform\+: Creates a platform in the game world.
\item Create\+Rope\+: Creates a rope in the game world.
\item Update\+Collider\+: Updates the colliders in the game world. 
\end{DoxyItemize}

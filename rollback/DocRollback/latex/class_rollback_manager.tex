\doxysubsection{Detailed Description}
\doxylink{class_rollback_manager}{Rollback\+Manager} is a class responsible for maintaining the integrity of the game simulation. 

It manages three game states with different temporalities\+:
\begin{DoxyItemize}
\item The current game state represents the local client\textquotesingle{}s real-\/time game.
\item The confirmed state is the last game state confirmed by the client (verified through checksum).
\item The state to be confirmed is the game state calculated by the master client once all inputs for a frame have been received. Other clients receive the checksum of this game state and verify its integrity against their own checksum for the same state. 
\end{DoxyItemize}

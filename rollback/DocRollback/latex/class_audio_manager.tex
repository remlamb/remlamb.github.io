\doxysubsection{Detailed Description}
Represents the \doxylink{class_audio_manager}{Audio\+Manager} responsible for managing audio in the game. 

The \doxylink{class_audio_manager}{Audio\+Manager} class handles loading, playing, and updating background music for the game. It provides methods for initialization, deinitialization, and updating the audio based on the current game state and volume settings.


\begin{DoxyItemize}
\item Variables\+:
\item is\+\_\+audio\+\_\+playing\+: Indicates whether audio is currently playing.
\item audio\+\_\+volume\+: The volume level of the audio.
\item music\+: The music stream object.
\item game\+\_\+logic\+\_\+\+: Pointer to the Game\+Logic object associated with the \doxylink{class_audio_manager}{Audio\+Manager}. 
\end{DoxyItemize}

\doxysubsection{Detailed Description}
Represents player input for a single frame. 

The \doxylink{class_input_1_1_frame_input}{Frame\+Input} class inherits from Custom\+Type and encapsulates input data for a specific frame, including the frame number and input flags.

Variables\+:
\begin{DoxyItemize}
\item frame\+\_\+nbr\+: The frame number associated with the input.
\item input\+: Flags representing player input actions for the frame. 
\end{DoxyItemize}

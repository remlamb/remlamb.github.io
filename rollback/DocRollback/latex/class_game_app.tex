\doxysubsection{Detailed Description}
Represents the main game application. 

The \doxylink{class_game_app}{Game\+App} class manages the main game loop, initialization, and deinitialization of various game components including game logic, rendering, audio, and networking. It also handles user interface interactions using Dear Im\+Gui for displaying menus and game options.

Variables\+:
\begin{DoxyItemize}
\item is\+\_\+collider\+\_\+visible\+\_\+\+: Indicates whether collider shapes are visible.
\item is\+\_\+game\+\_\+option\+\_\+visible\+\_\+ \+: Indicates whether the game options menu is visible.
\item app\+ID\+: The application ID for networking purposes.
\item app\+Version\+: The application version for networking purposes.
\item rollback\+\_\+manager\+: Manages game state rollback for network synchronization.
\item game\+\_\+logic\+: Manages the game logic and state.
\item game\+\_\+renderer\+: Handles rendering of the game world.
\item audio\+\_\+manager\+: Manages audio playback and volume settings.
\item network\+Logic\+\_\+\+: Handles networking logic and communication. 
\end{DoxyItemize}

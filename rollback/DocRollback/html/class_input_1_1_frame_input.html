Detailed Description

Represents player input for a single frame.

The FrameInput class inherits from CustomType and encapsulates input data for a specific frame, including the frame number and input flags.

Variables:

  • frame_nbr: The frame number associated with the input.
  • input: Flags representing player input actions for the frame.
